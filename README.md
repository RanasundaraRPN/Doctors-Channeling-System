# MediChannel

This project is separated into two main folders:

## Frontend

Located in `frontend/`.
To run the frontend:
1. Open a terminal in `frontend/`.
2. Run `npm install` (first time only).
3. Run `npm run dev`.

## Backend

Located in `backend/`.
To run the backend:
1. Open a terminal in `backend/`.
2. Run `npm install` (first time only).
3. Run `npm start`.

## Development

You need to run both frontend and backend terminals simultaneously for the full application to work.
The frontend runs on port 5173 (proxies API calls to port 5001).
The backend runs on port 5001.
